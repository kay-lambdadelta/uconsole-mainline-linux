This is a series of mainline kernel patches for the Clockwork Pi uConsole based upon the [PotatoMania/uconsole-cm3](https://github.com/PotatoMania/uconsole-cm3) patches.

WARNING: These patches are untested as my uConsole has not shipped yet, but they are largely the same as the one's that reportedly work. I will debug when it ships.

The device tree for the cm4 will be called "bcm2711-rpi-cm4-uconsole"
